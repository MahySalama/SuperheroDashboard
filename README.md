# SuperheroDashboard

A full‑stack dashboard built with **Django (backend)** and **React (frontend)**.  
Current features include API endpoints, pagination, search, filters, PostgreSQL integration, and a background data‑generation thread.

## Features (Current)
- Django REST API (synchronous backend)
- React dashboard UI with async axios calls
- Pagination (backend + frontend)
- Search & filtering
- Start/Stop background data generation thread
- Stats endpoint 
- PostgreSQL database

## Tech Stack
- **Backend:** Django, Django REST Framework  
- **Frontend:** React, Material UI  
- **Database:** PostgreSQL  
- **Tools:** Git, GitHub  

## Project Structure
GladProj/          # Django backend  
glad-frontend/     # React frontend  

## How to Run
### Backend
cd GladProj  
python manage.py runserver  

### Frontend
cd glad-frontend  
npm install  
npm start
